# Prompts

Prompt files and original prompt documents used for automated peer-review diagnosis.

## Main machine-readable prompt files

- `base_prompt.jsonl`: base review/meta-review prompts.
- `perturb_prompt.jsonl`: aspect-guided perturbation prompts.

## Original prompt documents

`original_prompts/` preserves the Word documents used during project development. Office metadata has been scrubbed from these `.docx` files in the release candidate.

## Recommended citation

If you reuse these prompts, cite the associated paper using `CITATION.bib` or the BibTeX block in `README.md`.
